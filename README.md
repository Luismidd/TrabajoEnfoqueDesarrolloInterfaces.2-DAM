# SmartOcupation – Consulta de Alquileres

Aplicación de escritorio desarrollada en Java que permite a los empleados de una inmobiliaria consultar el histórico de alquileres realizados por los clientes en un rango de fechas determinado.

La aplicación muestra la información del alquiler, del cliente y de la vivienda, y genera un informe visual con los resultados obtenidos.

## Funcionalidades
- Selección de fecha de inicio y fecha de fin mediante JDateChooser
- Consulta del histórico de alquileres desde una base de datos MySQL
- Visualización de los resultados en un informe con JasperReports
- Gestión de errores mediante cuadros de diálogo

## Tecnologías utilizadas
- Java
- Java Swing
- JDBC
- MySQL
- JasperReports
- JCalendar (JDateChooser)
- Maven
- NetBeans

Proyecto realizado con fines académicos.
